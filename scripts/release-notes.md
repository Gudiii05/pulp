Primera versión pública de Pulp.

## Qué incluye

- **Captura automática del portapapeles** — texto, código e imágenes.
- **Clasificación automática** por tipo (texto, código, imagen) con pestañas para filtrar.
- **Atajo global configurable** — por defecto `Ctrl + Shift + V`, lo cambias en Ajustes.
- **Fijar copias favoritas** para que no se borren nunca.
- **Búsqueda en tiempo real** dentro del historial.
- **Modo claro y oscuro** con cambio manual.
- **Diseño glassmorphism** inspirado en macOS.
- **System tray** con menú rápido (Abrir / Limpiar todo / Salir).
- **Actualizaciones automáticas** desde GitHub Releases.
- **100% local** — toda la información se guarda en una base SQLite local, no se envía nada a ningún servidor.

## Instalación

Descarga el archivo `Pulp_0.1.0_x64-setup.exe` y haz doble clic. Si Windows SmartScreen avisa de un editor desconocido, haz clic en "Más información" → "Ejecutar de todas formas".

Más detalles en el [README](https://github.com/Gudiii05/pulp/blob/main/README.md).
