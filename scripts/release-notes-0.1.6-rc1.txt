## Smoke test — first run of the new CI release pipeline

This is a pre-release (`-rc1`) for validating the new GitHub Actions release workflow. **Do NOT install over a production Pulp install.**

### What's being validated

- Dual-platform build on Windows + Linux from a single tag push
- Tauri updater signing on both platforms via GitHub Actions secrets
- `latest.json` assembly from both platforms' `.sig` files
- Linux `.deb` runtime deps declaration (`libwebkit2gtk-4.1-0`, `libayatana-appindicator3-1`)
- AppImage with bundled GStreamer media framework

### Known limitations on Linux

See [Limitaciones en Linux](https://github.com/Gudiii05/pulp/blob/main/README.md#limitaciones-en-linux) — GNOME Wayland is not fully supported (global hotkey, tray icon, and background clipboard capture have constraints).

If you encounter issues on this rc1, please open an issue with the platform (Windows / Ubuntu version / desktop environment / X11 or Wayland).
